回归表的四个区域

表头

说明因变量、模型编号和每一列的设定。

变量行

每一行对应一个解释变量或控制变量。

系数与括号

第一行通常是系数；括号内通常是标准误。

表尾

报告样本量、R²、控制变量和标准误类型。

阅读顺序先确认因变量和模型列，再读系数与不确定性，最后检查表尾。

OLS 基础

OLS 的基本语言

从数据、变量和线性关系开始，逐步说明系数、预测值、残差和最小二乘。

OLS 基础

数据、观测值与变量

数据（data）

研究中记录的全部信息。本文的数据来自 203 名受访者。

观测值（observation）

数据表中的一行。本文每一行对应一名受访者。

变量（variable）

数据表中的一列，例如年龄、社会距离或环保态度。

例子受访者 17 的社会距离得分、年龄和环保态度共同构成一条观测。
OLS 基础

因变量与自变量

Y

因变量（dependent variable）

研究需要解释或预测的结果。本文示例使用总体环保态度的标准化合成得分。

X

自变量（independent variable）

用于解释 Y 差异的变量。这里使用社会距离得分。

研究问题社会距离较高的受访者，其环保态度平均是否较低？
OLS 基础

线性关系与条件平均值

线性关系（linear relationship）

用一条直线概括 X 与 Y 的平均变化趋势。

条件平均值（conditional mean）

给定某个 X 水平时，Y 的平均水平。回归线描述的就是这个平均值。

OLS 基础

预测值、误差项与残差

预测值（predicted value）

Ŷᵢ 是回归线在第 i 个受访者 X 值处给出的平均预测。

误差项（error term）

εᵢ 表示总体模型中未被 X 解释的部分，研究者无法直接观察。

残差（residual）

eᵢ = Yᵢ − Ŷᵢ，是样本中观测值与预测值的差。

区别误差项属于未知总体模型；残差由当前样本计算得到。
OLS 基础

残差平方和与拟合线

1.00

残差平方和（SSE，示意）

比较各条线的 Σeᵢ²。数值最小的线被选中。

OLS 基础

普通最小二乘法（OLS）

普通（ordinary）

最基础的线性回归估计方法。

最小（least）

在所有候选系数中寻找目标函数的最小值。

二乘（squares）

目标函数是残差的平方和 Σeᵢ²。

平方的作用正负残差不会相互抵消；较大的残差会受到更高权重。
OLS 基础

简单线性回归方程

Yᵢ = α + βXᵢ + εᵢ
Yᵢ第 i 人的结果
α截距
β斜率或系数
Xᵢ第 i 人的解释变量
εᵢ未解释部分
拟合方程估计完成后使用 Ŷᵢ = α̂ + β̂Xᵢ 计算样本内预测值。
OLS 基础

截距（intercept）

α̂

X = 0 时的预测值

截距确定回归线在纵轴上的起点。

有时具有实际意义

若 X = 0 是可观察且合理的状态，截距可以解释为该状态下的平均 Y。

有时只是计算起点

若样本中没有 X = 0，截距仍参与拟合，但缺少直接的现实解释。

OLS 基础

回归系数（coefficient）

定义在简单线性回归中，X 增加 1 个单位时，Y 的条件平均值变化 β 个单位。

方向

β > 0 表示同向变化；β < 0 表示反向变化。

大小

|β| 表示 X 每增加 1 单位时，Y 平均变化多少。

单位

系数单位由 X 和 Y 的计量单位共同决定。

OLS 基础

标准化得分（standardized score）

计算

z = (原始值 − 均值) ÷ 标准差

标准化后，样本均值约为 0，标准差约为 1。

系数单位

本文因变量采用多个题项的标准化均值。因而 −0.230 表示环保态度约下降 0.230 个标准差。

它不表示下降 23%，也不表示 Likert 原始分数下降 0.230 分。

用途标准化便于比较原始量纲不同的题项与量表。

估计值的不确定性

样本系数会随抽样变化。标准误、检验和置信区间用于描述这种变化。

统计推断

总体与样本

总体（population）

研究结论希望适用的完整群体。总体中的关系通常无法被完整观察。

样本（sample）

实际进入数据并参与估计的个体。本文样本为 203 名意大利 Prolific 受访者。

● ● ● ● ●
外推限制在线样本在性别、年龄和地区上做了平衡，但不能自动代表所有意大利居民。

总体参数与样本估计值

β：总体参数（parameter）

总体中的固定关系。它通常未知，需要借助样本估计。

β̂：估计值（estimate）

使用当前样本计算得到的数值。更换样本后可能变化。

估计量（estimator）

从样本数据计算 β̂ 的规则。OLS 就是一种估计规则。

中心位置

分布宽度

定义标准差描述一组观测值围绕其均值的典型距离。

标准差较小

统计推断

标准误（standard error）

定义标准误是估计量抽样分布的标准差，用于描述估计值在重复样本中的波动程度。

SE 较小

不同样本得到的 β̂ 较集中，点估计较精确。

SE 较大

不同样本得到的 β̂ 较分散，点估计较不精确。

统计推断

标准差与标准误的区别

标准差（SD）

描述个人数据的分散程度。对象是 Yᵢ、Xᵢ 等观测值。

标准误（SE）

描述系数估计的抽样波动。对象是 β̂ 等估计值。

共同点

二者都衡量分散程度，但研究对象和用途不同。

例子受访者态度差异较大，并不必然意味着社会距离系数的标准误也很大。
统计推断

影响标准误的因素

基准抽样分布

结果解释

选择一种情形，观察抽样分布宽度。

零假设与备择假设

H₀

零假设（null hypothesis）

β = 0。总体中不存在社会距离与环保态度的线性关联。

H₁

备择假设（alternative hypothesis）

β ≠ 0。总体中存在正向或负向的线性关联。

检验逻辑先在 H₀ 成立的条件下建立参考分布，再判断当前样本结果是否异常。

t = (β̂ − 0) ÷ SE(β̂)

|t| 较大

系数相对标准误较大，与 H₀ 的距离较远。

|t| 较小

系数相对标准误较小，样本证据较弱。

p 值（p-value）

定义在 H₀ 和模型假设成立时，获得当前统计量或更极端统计量的概率。

条件

p 值的计算以 H₀ 成立为前提。

对象

含义

p 较小表示当前数据与 H₀ 的相容程度较低。

显著性水平与第一类错误

显著性水平（α）

研究者在检验前设定的判断阈值。常用 α = 0.10、0.05 或 0.01。

p < α → 拒绝 H₀

第一类错误（Type I error）

总体中 H₀ 实际成立，但样本检验拒绝了 H₀。α 控制这种长期错误率。

一次显著结果仍可能出错，因此需要研究设计、重复研究和稳健性检查。

多元回归同时纳入多个解释变量，用条件比较描述各变量与结果之间的关系。

定义控制变量是与研究问题相关、被同时纳入回归模型的其他观测变量。

多元回归

R²（决定系数）的定义

定义R² 表示模型在当前样本中解释的 Y 总变异比例。
R² = 1 − 未解释变异 ÷ 总变异

总变异（SST）

Y 与样本均值之间的平方差总和。

未解释变异（SSE）

Y 与回归预测值之间的残差平方和。

已解释变异

SST − SSE。

多元回归

R² = 0.217 与调整后 R²

0.217

样本内拟合程度

完整模型解释了当前样本中总体环保态度约 21.7% 的变异。

它不表示模型解释了 21.7% 的受访者，也不证明因果关系。

调整后 R²（adjusted R²）

加入变量通常不会降低普通 R²。调整后 R² 对变量数量施加惩罚，更适合比较复杂度不同的模型。

预测目标和解释目标对 R² 的要求也可能不同。

多元回归

OLS 的基本假设与关注点

线性设定

条件平均关系能被所设方程合理描述。

独立抽样

一名受访者的误差不系统决定另一人的误差。

无完全共线

解释变量不能由其他变量完全复制。

条件均值为零

误差项不能与解释变量系统相关；该条件关系到无偏与因果解释。

标准误同方差影响传统标准误公式；异方差稳健标准误可放宽这一要求。

可以修正

无法修正

  • 遗漏变量
  • 反向因果
  • 测量误差
  • 样本选择偏误

观察性回归的解释边界

反向因果

环保态度较强的人可能主动获取气候信息，因而感知距离较近。

遗漏变量

开放性人格可能同时影响心理距离和环保态度。

Social distance 题项原句表示“影响我、家人与朋友”。同意程度越高意味着距离越近，因此编码方向需要反转。

反向分数 = 6 − 原分数

标准化均值

将题项转成共同尺度后再取平均。最终量表的 0 附近表示样本平均水平，正负值表示相对高低。

Scale = mean(z₁,…,zₖ)
读系数前先确认每个变量的高分代表什么。编码方向决定系数符号的语言解释。

测量与模型

有界、删失与截断

有界（bounded）

测量工具规定最小值和最大值，例如 1–5 题项。潜在值未必存在界外部分。

删失（censored）

界外个体仍在数据中，但观察值被记录为阈值，例如收入超过上限统一记为上限。

截断（truncated）

界外个体不进入样本，研究者看不到其观测记录。

模型含义变量有上下界只说明 bounded；Tobit 需要潜在连续变量和 censoring 机制。
测量与模型

Tobit 模型及本文的稳健性分析

潜在结果 Y*

Y* = Xβ + ε

当 Y* 位于上下限之间时观察其数值；越过阈值时观察值被记录为端点。

Tobit 系数首先描述 X 对潜在 Y* 的关系，观测 Y 的边际效应需要另行计算。

本文的使用方式

论文依据合成量表的端点集中（corner solutions）报告 Tobit，并比较 OLS 与 Tobit 的方向和显著性。

解释时仍需评估潜在连续性、删失机制、正态性与同方差假设是否合理。

结论Tobit 为本文提供模型替代检验；它不构成所有有界合成量表的通用选择。
